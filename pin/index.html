Data

Data is available from the GitHub repository and archived on Zenodo for citability.

The repository contains GPIN and OWR estimates for 2003–2024 (~60,000 stock-year observations per model):

  • data/gpin_all.csv — GPIN model estimates
  • data/owr_all.csv — OWR model estimates

Based on WRDS DTAQ Intraday Indicators. CRSP shrcd 10, 11 and exchcd 1, 2, 3, 4.

If you use these estimates in your work, please cite our paper:

Duarte, Jefferson, Edwin Hu, and Lance Young. 2020. "A Comparison of Some Structural Models of Private Information Arrival." Journal of Financial Economics 135(3): 795–815. doi:10.1016/j.jfineco.2019.08.005

Code

All code and data are available on GitHub: edwinhu/pin-code

The repository includes:

  • Estimation models: gpin_model.py, owr_model.py, est.py
  • SAS data pipeline for WRDS/DTAQ processing
  • Slurm and SGE job scripts for cluster estimation
  • Pre-computed estimates: data/gpin_all.csv and data/owr_all.csv (2003–2024)
